Function arrGet

Get the value of an array at the given index, if the index is not found the return value is based upon the given $return_value

Author: Thomas Mundal <thmundal@gmail.com>
Located at util.php
Parameters summary
array $array The array from which the value is to be gotten.
mixed $key The key to the value to be gotten.
mixed $return_value = "%throw-exception%" Default value is "%throw-exception%" and the function will throw an exception if the key could not be found in the array and no return value is specified.
Return value summary
mixed Returns the data at the key in the given array
Thrown exceptions summary
Exception Exception is thrown if the given key does not exist in the array, and when a return value is not specified
